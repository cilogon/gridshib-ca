GridShib CA Python Client

To download the GridShib-CA python client, right click on the following link and select "Save link as..."

GridShib-CA python client

Your authentication token is { $authToken }
Remember to keep your token private.

You can run the python client as follows: 

 python gridshibca-client.py -a { $authToken }

If you want, you can leave the -a option and token off of the commandline and you will be prompted for it.

Note that you will need to have pyOpenSSL installed to run this client.
